学习 Class Arrays

Chaoyue's Notes
2019-07-17
技术总结

Docs

Class Arrays

Method Summary

  • asList

返回固定大小List

  • binarySearch

可以在整个数组中查找，也可以在一定范围内查找

支持Long/int/short/double/char/byte/float/object/T类型数组的查找

  • copy & copyRange
  • sort

升序排列

另外，Arrays的sort方法也可以结合比较器，完成更加复杂的排序。

  • toString

返回数组内容的一个String

  • deepToString

打印多维数组（multidimensional arrays）的内容

  • equls

比较两个一维数组是否相等

  • deepEquals

判断更复杂的数组是否相等

  • fill

向整个数组或一定范围内的数组填充指定内容
