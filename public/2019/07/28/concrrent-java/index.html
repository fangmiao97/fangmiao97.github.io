CAS

CountDownLatch

一个线程等待其他活动都完成后才执行

使用构造函数，声明需要等待多少个活动

使用countdown将这个要等待的活动数目减少

调用await()方法的线程会被挂起，它会等待直到count值为0才继续执行

Semaphore

acquire申请一个许可

release释放一个许可
